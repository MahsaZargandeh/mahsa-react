# Golestan Tea React Website

A modern responsive tea shop website built with React and Tailwind CSS.

## Features

- Responsive Design
- Mobile Hamburger Menu
- Product Slider with Swiper
- Smooth Scroll Animations
- Product Cards
- Contact Page
- React Router Navigation
- Modern UI Design


## Technologies

- React
- Vite
- Tailwind CSS
- Framer Motion
- Swiper
- React Router


## Installation

npm install

npm run dev
