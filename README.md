# 🌿 Golestan Tea Website - React

A modern and responsive tea shop website built with React and modern frontend technologies.

This project is a frontend implementation of a tea brand website with a clean UI, responsive design and interactive components.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/MahsaZargandeh/golestan-tea-react.git
```

Go to project folder:

```bash
cd golestan-tea-react
```

Install dependencies:

```bash
npm install
```

Run project:

```bash
npm run dev
```
